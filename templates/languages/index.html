Documentación y Notas

Este sitio web fue desarrollado con la finalidad de poner en práctica mi conocimiento y así desenvolverme más en el manejo de django. La premisa es construir proyectos que puedan resultarme útil en algún momento dado.

Funciona sobre un solo archivo html, cuya estructura fue diseñada para que se muestre de manera dinámica el contenido solicitado. Mi intención con este proyecto era trabajar las consultas a la base de datos haciendo buen uso de los querys y aplicar habilidades basícas de frontend para que el contenido pueda verse desde cualquier tipo de pantalla.

Su utilidad es la de un blog de notas, pues fue creado para ir guardando puntos que me resulten claves de las tecnologias que voy aprendiendo, de manera tal que pueda acceder a estos de una forma facíl en el momento que lo requiera.
